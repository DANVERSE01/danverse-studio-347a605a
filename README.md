# DANVERSE — AI-Powered Creative Studio

**Domain:** [danverse.ai](https://danverse.ai)

## About

DANVERSE is an AI-powered creative studio at the intersection of technology and culture. We deliver branding, advertising, web/digital, 3D/immersive, and AI systems work for forward-thinking clients.

## Tech Stack

- **Framework:** React + TypeScript
- **Build:** Vite
- **Styling:** Tailwind CSS + shadcn/ui
- **Animation:** Framer Motion + Lenis smooth scroll
- **Deployment:** Netlify (SPA)

## Development

```sh
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Deployment

The project deploys to Netlify. Push to `main` to trigger a production build. See `netlify.toml` for configuration.
